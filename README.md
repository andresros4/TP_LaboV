# TP_laboV
Aplicación Android desarrollada como trabajo práctico de la materia Laboratorio V.
Es una app sencilla, que consume la api de TheCocktailDB, mostrando una lista con todos los cócteles, con su respectiva información (foto, nombre, ingredientes) y al clickear en alguno de ellos se muestra cómo prepararlos. También permite buscar un cóctel por su nombre y muestra toda la información del mismo.
